ASSETS FOLDER

This folder is for storing website assets:

1. cv.pdf - Add your actual CV PDF file here
2. favicon.ico - Add a website favicon (optional)
3. profile.jpg - Add your profile photo (optional)

To add your CV:
- Save your CV as "cv.pdf" in this folder
- The download link in the website will automatically work
- Make sure the PDF is optimized for web (under 5MB recommended)

To add a profile photo:
- Save your photo as "profile.jpg" in this folder
- Update the HTML to use the actual image instead of the placeholder
- Recommended size: 300x300 pixels or larger
